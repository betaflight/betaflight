## HW info

* STM32F722RET6 64lqfp 216MHz
* MPU6000 SPI
* MS5611 baro
* 4 uarts available + VCP
* 10 pwm outputs + 6 inputs
* external I2C
* external SPI (shared with U4/5)
* 128Mb dataflash logging (SPI)
* Voltage measurement, with 10k/1k divider
